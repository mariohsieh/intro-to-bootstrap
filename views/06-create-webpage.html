Here are some hints to help the development process:

  • Before you start writing code, take some time to analyze and breakdown the mockup
  • Using HTML5 semantic markup, determine where you would use header, main, section, article, etc...
  • Keeping the grid system in mind, organize the content by rows and columns
  • Once you have the page layout down, write out the HTML content
  • Don't spend too much time on CSS styling until after the HTMl content has been written
  • After the major elements are in place, then start tinkering with styling

The following pages will give a step-by-step tutorial on creating the website.  However, I encourage you to try and create the website on your own first and if you get stuck, then refer back to this lesson plan for help.

Good luck and take your time.  You can also try some pair programming and work on this collaboratively with another student.  If you get stuck, try researching the answer online first before you ask a TA/teacher for help.
