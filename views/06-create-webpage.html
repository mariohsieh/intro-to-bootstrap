Here are some hints to help the development process:

  • Before you start writing code, take some time to analyze and breakdown the mockup
  • Using HTML5 semantic markup, determine where you would use header, main, section, article, etc...
  • From there, organize the content by Bootstrap rows and columns
  • Once you have the page layout down, write out the HTML content
  • Don't spend too much time on CSS styling unil after the HTMl content has been written
