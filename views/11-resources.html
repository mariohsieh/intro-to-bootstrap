Congratulations!

Job well done on completing the Introduction to Bootstrap curriculum but you shouldn't stop here.  There are many more features that Bootstrap has to offer.  We encourage you to research and improve your skillset even more with the following resources.

Helpful Resources

  • The official Bootstrap homepage has a very thorough documentation.
  • Mozilla Developer Network is probably the largest, most comprehensive source of front end documentation.
  • DocHub is another comprehensive resource.
  • A List Apart is an online magazine with great content on front end best practices.
  • Smashing Magazine is oriented toward web designers, but they frequently have good, short how-to articles on CSS and JavaScript coding.
  • HTML5 Rocks is a great site for keeping up on the latest advancements in front end techniques and technology.
