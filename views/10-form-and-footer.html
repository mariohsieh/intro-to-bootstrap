Form & Footer

Goals

  • Finish up the remaining main content

  • Write the code and style the footer

Overview

There is one more row of the main content to code then we'll finish up with the footer.

Step 1

Again, referring to the mockup, you will see the last row of the main section has a single 4-unit wide column centered on the webpage.  This column holds a submission form that contains two input fields for a name and address, a textarea for comments, and a submission button.  Also, a label for each respective form element.

So let's code out the column: 


<main>
  ...
  <section class="row">
    <article class="col-md-4 col-offset-md-4">
      ...
    </article>
  </section>
</main>

Then fill in the form: 


<main>
  ...
  <section class="row">
    <article class="col-md-4 col-offset-md-4">
      <h3>Contact Us</h3>
      <form>
        <div class="form-group">
          <label for="fullName">Full Name:</label>
          <input type="text" id="fullName" class="form-control">
        </div>
        <div class="form-group">
          <label for="emailAddress">Email Address:</label>
          <input type="text" id="emailAddress" class="form-control">
        </div>
        <div class="form-group">
          <label for="comments">Comments:</label>
          <textarea id="comments" class="form-control">
        </div>
      </form>
    </article>
  </section>
</main>

Step 2

With the header and main tags done, we can finish this webpage by coding the footer.  The footer content consists of one column, 6-unit in width, and two elements nested inside. A hr line and a copyright for Railsbridge that is also centered inside the column.

Let's write the code for the footer. 


<footer>
  <article class="col-md-6 col-offset-md-3">
    <hr>
    <p class="text-center">© 2014 RailsBridge</p>  
  </article>
</footer>

To finish it up, please adjust CSS for the HTML tags if they are not displaying properly.
