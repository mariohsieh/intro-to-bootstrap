Bootstrap Basics

Goals

  • Practice adding Bootstrap classes

  • Change text and background color of simple HTML elements

Overview

Using Bootstrap is as simple as adding classes to your HTML elements. The Bootstrap CSS file contains numerous class definitions with pre-set styles and all you have to do is add those classes to the HTML element that you wish to style.  So you have some control over how much Bootstrap styling your web application has as you decide what classes to add or not.

Having said that, there are a few HTML tags that are automatically Bootstrap-styled without adding any classes such as the <a> tag and all heading tags <h1>...<h6>.

Steps

Step 1

Let's start with styling some basic <p> tags by changing its text color.

Bootstrap has a set of contextual colors that are used throughout the framework.  Take a look at the keywords and their corresponding colors below:

  • muted

  • primary

  • success

  • info

  • warning

  • danger

To change the text color of a <p> tag, we use the prefix: "text-" with a keyword and then add it to the class attribute.  For example, to change the text color to "green", we add "text-success" as one of the classes.  Or to change it "red", we add "text-danger".

In the input box below, practice changing the text color of the example sentence:


<p class='{{colorChange}}'>Hello there!  How are you?</p>

Hello there!  How are you?

In addition to changing the text, we can also change the background color with the same keywords. Instead of the prefix "text-" use "bg-" to change the background color. (Note:  There is no "bg-muted")

In the same input field above, mix and match different combinations of text/background colors by adding both classes separated by a single whitespace.

Step 2

Now that you have some practice under your belt, write out some <p> tags in your index file and style them using the bootstrap classes from above.  Try styling some <h1>,<h2>,...,<h6> tags as well.
