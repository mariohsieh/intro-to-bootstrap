Bootstrap Basics

Goals

  • Practice adding Bootstrap classes to HTML elements

Overview

Using Bootstrap is as simple as adding classes to your HTML elements. The Bootstrap CSS file contains numerous class definitions with pre-set styles and all you have to do is add those classes to the HTML element that you wish to style.  So you have some control over how much Bootstrap styling your web application has as you decide what classes to add or not.

Having said that, there are a few HTML tags that are automatically Bootstrap-styled without adding any classes such as the <a> tag and all heading tags <h1>...<h6>.
