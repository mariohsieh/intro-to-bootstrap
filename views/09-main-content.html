Main Content

Goals

  • Identify what HTML tags are needed for the content.

  • Write the HTML code and add Bootstrap classes for columns

  • Style the CSS properties to resemble mockup

Overview

Continue re-creating the mockup by coding out the main content. Using the mockup with the columns overlayed from the last page, code the first two sections of the main content.  Then, write the HTML code for the images, list, and the large quote by Tony Bennett.

Step 1

The first row of the main content contains three images.  Before writing anything, create a sub-folder called "img" in your project folder and put the images inside.  Now refer back to the mockup to see the widths needed for each column:

The columns for the row of images should like this: 


<main>
  <section class="row">
    <article class="col-md-2 col-offset-md-3">
      ...
    </article>
    <article class="col-md-2">
      ...
    </article>
    <article class="col-md-2">
      ...
    </article>
  </section>
...

With the columns set, write in the html code for the images. 


<main>
  <section class="row">
    <article class="col-md-2 col-offset-md-3">
      <img alt="Bay Bridge" src="img/bay-bridge.jpg">
    </article>
    <article class="col-md-2">
      <img alt="Palace of Fine Arts" src="img/palace-of-fine-arts.jpg">
    </article>
    <article class="col-md-2">
      <img alt="Legion of Honor" src="img/legion-of-honor.jpg">
    </article>
  </section>
...

You may have noticed that the images don't quite fit.  What can you do with CSS to make them display properly?  At this point, adjust the CSS properties to resemble the mockup.

Step 2

Let's move on the next row of the main content.  The left column nests a single unordered list and the right column has one large line of text followed by a smaller line of text.

Referring to the same mockup from before, let's write out the code for the columns: 


<main>
  ...
  <section class="row">
    <article class="col-md-2 col-offset-md-3">
      ...
    </article>
    <article class="col-md-4">
      ...
    </article>
  </section>
...

Then fill in the content. Notice that the second line of text is aligned to the right, so you'll have to add a text-right class. 


<main>
  ...
  <section class="row">
    <article class="col-md-2 col-offset-md-3">
      <ul>
        ...
      <ul>
    </article>
    <article class="col-md-4">
      <h2>"I left my heart in San Francisco."<h2>
      <h3 class="text-right">-Tony Bennet</h3>
    </article>
  </section>
...

With the columns set and the content in place, now just style any remaining display issues with CSS to resemble the mockup.
