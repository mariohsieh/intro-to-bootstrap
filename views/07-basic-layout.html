Basic Layout

Goals

  • Identify general areas of website

  • Write skeleton HTML code

Overview

Before writing any code look at the mockup and determine the general areas of the webpage.  Using HTML5 tags, where would you use the header, main, and footer tags to breakdown the mockup?  Once you have determined that, write out the major areas on a new html page for the skeleton code.

Steps

Step 1

Since Bootstrap uses a grid system of rows and columns, I would organize my HTML accordingly as well.  First identify the majors areas from top to bottom first, then figure out the columns within those areas from left to right.  I have overlayed the mockup with colored blocks to mark the major areas:

Using the image above, let's write out the skeleton code for our webpage: 


<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>Discover San Francisco</title>
  </head>
  <body>
    <header>

    </header>
    <main>

    </main>
    <footer>

    </footer>
  </body>
</html>

Step 2

Now that the general layout is done. Let's determine where to create Bootstrap rows:

The HTML code should look like this:


<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>Discover San Francisco</title>
  </head>
  <body>
    <header>
      <section class="row">

      </section>
    </header>
    <main>
      <section class="row">

      </section>
      <section class="row">

      </section>
      <section class="row">

      </section>
    </main>
    <footer>
      <section class="row">

      </section>
    </footer>
  </body>
</html>
