Forms Elements

Goals

  • Styling form elements such as buttons, inputs, selects, and text areas

Overview

In this lesson, we'll style HTML elements that are associated with forms. Starting with buttons, we'll work our way through the different input tags, selects and textareas. By the end of this lesson, you'll be able to create a submission form with Bootstrap style.

Steps

Step 1

Using Bootstrap buttons will require adding the class btn.  From here, you can adjust the size of the buttons, change the color, and alter its active state.

Bootstrap has four sizes of buttons:

  • x-small

  • small

  • default

  • large

We can set the sizes by adding an additional class: btn-xs, btn-sm, and btn-lg. The default size does not require any additional classes.

Buttons can also be stylized with contextual colors. The prefix for buttons is btn-.

Buttons are displayed inline by default but we can make them block-level by adding btn-block.

Step 2

Let's move on to input fields.

All text-based input fields can be styled with the class: form-control.  A text-based Bootstrap input field will look like this:

An input field is usually accompanied a label.  We can group a label with its respective input field by nesting them under a div. This particular div will have the class form-group.

The following example login form demonstrates this grouping: 


<form>
  <div class="form-group">
    <label for="usernameInput">Username</label>
    <input id="usernameInput" class="form-control" />
  </div>
  <div class="form-group">
    <label for="passwordInput">Password</label>
    <input id="passwordInput" class="form-control" />
  </div>
  <button type="submit" class="btn btn-lg btn-primary">Login</button>
</form>
