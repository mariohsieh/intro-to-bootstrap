Introduction to Bootstrap

Goal

This curriculum provides an introduction to the front-end framework Bootstrap.  In the lesson plans ahead, the following topics will be covered:

  • Styling HTML elements with classes
  • Building layouts with the grid system
  • Creating a webpage from mockup

By the end of this course, you will have a solid foundation in the Bootstrap framework and be comfortable creating layouts with the grid system.

Schedule

  • 30 minutes to set up Bootstrap and to get familiar with the basics
  • 90 minutes of the grid system, styling HTML elements, and making tables/lists/forms
  • 1 hour to build a web page

Of course, the schedule above is just a guideline. Please feel free to learn at your own pace.

Requirements

You will need the following for this curriculum:

  • Completion of the Front-End Workshop
  • Web browser: Any web browser will work as long as you are comfortable with the developer tools.  Having said that, we recommend using Google Chrome.
  • Text Editor: Again, any text editor should be fine but one with syntax highlighting is preferable.  As for recommendations, GitHub's atom.io for Windows/Mac, and Geany for Linux.

Working Effectively and Efficiently

We highly recommend you do the following:

  • Open your browser fresh or hide any windows you already have open.
    • Bring up one window with two tabs
    • One for this content
    • One for interacting with your app
  • Open your text editor and do not ever close it. We're not quitters.
  • Hide all extra applications. Turn off twitter, IM, and all other distractions.

By minimizing the number of things you interact with, you reduce the amount of time spent switching between them and the context lost as you work through the lessons. Having 50 tabs open in your web browser gets confusing and wastes time.
