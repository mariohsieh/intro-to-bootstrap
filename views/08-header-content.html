Header Content

Goals

  • Add Bootstrap classes for layout

  • Write content for header

  • Style CSS properties to resemble mockup

Overview

Write all the header content first before worrying about the CSS stylings. Then proceed to add Bootstrap layout classes so the elements are displayed in the right position. Once those are set, add any additional CSS styles to get the right look.

Steps

Step 1

Using the same mockup with the rows overlayed, I have overlayed yellow boxes to represent columns over the previously used mockup.

Looking at the header, there are 3 columns so let's write the skeleton HTML code for that in the header: 


<header>
  <section class="row">
    <article class="">

    </article>
    <article class="">

    </article>
    <article class="">

    </article>
  </section>
</header>

Now we need to add the Bootstrap classes to make the article tags actual columns.  Notice that the columns are centered but since we used container-fluid we have to offset the first column.

The 3 columns are taking up 75% of the total horizontal space so if there are 12 total units then 75% of 12 units would be 8 units for the 3 columns.

12 - 8 leaves 4 units for the margin on the left and right.  Since we only need to worry about the left margin, we will set the offset value to 2.

Look at the image below to see how to divide the 12 units across the rows.

So let's fill in the columns and also the HTML code for the heading, input box and the search button: 


<header>
  <section class="row">
    <article class="col-md-5 col-offset-md-2">
      <h1>San Francisco - <span>The city by the bay</span></h1>
    </article>
    <article class="col-md-3">
      <input type="text" class="form-control" placeholder="Search this site">
    </article>
    <article class="col-md-1">
      <button class="btn btn-default">Search</button>
    </article>
  </section>
</header>

Step 2

For this step, we want you to style the HTML tags of the header to resemble the mockup. Some of text may not be aligned properly or the font might be the wrong size which means you'll have to adjust the CSS properties to get them to display correctly.  The vertical-align and font-size properties might be helpful to solve these issues.
