Grid System

Goals

  • Containers, rows, and columns of the grid system

  • Column offsets and ordering

Overview

This lesson will cover one of Bootstrap's most powerful features, the grid system. First we will go over the container class and then move onto page layouts using rows and columns. From there, we'll breakdown the 12-column grid system and how to center, offset or re-order the columns.

Steps

Step 1

Bootstrap offers two types of container classes: container for centered fixed-width content, or container-fluid for full-width content.  Look at the two images below for comparison.  The red area is where your content would appear.

If you set an HTML element with class container, all the nested content inside will be confined to the horizontal boundaries set by the container.

Rows and Columns

Page layout is handled by defining an HTML element (usually a div but can also use section, article, aside, etc...) of class row and nesting a column(s) underneath as a direct child(ren). 


<div class="row">
  column
  column
  column
</div>

NOTE: not actually how to write columns, just placeholder text

Columns are created based on the concept that there are 12 units across the horizontal space in a row and any one column can have a width between 1 and 12 units.  If there are multiple columns in a row, any column that exceeds the maximum 12 allowable units will wrap onto the next line.

To create a column, we define an HTML tag with a class of col-XX-Y where XX is the screen size and Y is the number of units for the width of the column (1 to 12).

Here is the breakdown for the different screen sizes Bootstrap uses:

Screen Size Code to write Screen Width
Extra Small xs less than 768px
Small sm 768px and up
Medium md 992px and up
Large lg 1200px and up

Now let's take a look at some examples:
(The following columns have been styled with a background-color color and height: 300px to see the column dimensions better.  Also, container is in use)

  </div>

Three columns of equal width


<section class="row">

  </div>
  <div class="col-md-4">
    <!-- second column content here-->
  </div>
  <div class="col-md-4">
    <!-- third column content here-->
  </div>
</section>
