Getting Started

Goals

  • Load Bootstrap onto your HTML file

  • Practice styling with Bootstrap classes

Overview

What is Bootstrap?

An open-sourced project originally developed by two employees at Twitter, Bootstrap has grown to become the most popular front-end framework used for responsive and mobile web applications.

As of August 2014, it has over 70k stars and 26k forks on GitHub.

Steps

Step 1

Create a project folder on your computer and call it "railsbridge-bootstrap-course". Now open your text editor and save a blank html file in this project. You can call it "index.html". Fill out this HTML file with the basic skeleton code or just copy and paste the example template below: 

<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">>
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>RailsBridge - Bootstrap Course</title>
  </head>
  <body>
    <!-- content here -->

  </body>
</html>

Step 2

Now we need to load the Bootstrap framework onto your HTML file. There are a few ways to do this but the easiest method is to link to the Bootstrap CDN by putting the code below inside the <head> tag: 


<link rel="stylesheet" href="//maxcdn.bootstrapcdn.com/bootstrap/3.2.0/css/bootstrap.min.css">

You can also download Bootstrap here so you have a local copy on your computer. Unzip the files and save the "bootstrap.min.css" or "bootstrap.css" file into a CSS folder inside your project folder and link to that css file inside the <head> tag: 


<link rel="stylesheet" href="css/bootstrap.min.css">
