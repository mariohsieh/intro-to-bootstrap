Forms & Buttons

Goals

  • Styling buttons

  • Styling inputs, textareas, and selects

Overview

In this lesson, we'll style HTML elements that are associated with forms. Starting with buttons, we'll work our way through the different input tags, selects and textareas. By the end of this lesson, you'll be able to create a submission form with Bootstrap style.

Steps

Step 1

Using Bootstrap buttons will require adding the class "btn".  From here, you can adjust the size of the buttons, change the color, and alter its active state.

Bootstrap has four sizes of buttons:

  • extra small

  • small

  • medium

  • large

We can set the sizes by adding an additional class: btn-xs, btn-sm, and btn-lg. The default size does not require any additional classes. To create a large button, the code would look like this: 

<button class="btn btn-lg">

Buttons, much like paragraphs and headings, can be stylized with contextual colors. There is one difference however, in that for the grey color, you have to use 'default' instead of muted. All the other keywords will work. The prefix for buttons is "btn-".
