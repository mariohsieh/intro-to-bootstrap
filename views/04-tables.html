Tables

Goals

  • Add different classes to style tables

  • Style rows with hover states and contextual colors

Overview

This lesson we'll go through the different table styles that Bootstrap offers.  In addition, we'll see how to style rows with contextual colors.

Steps

Step 1

Tables can look much better with only a couple added Bootstrap classes.  A basic table with no added CSS looks like this:

By adding one class table: 


<table class="table">
  ...
</table>

It can look like this:

We can create a bordered table: 


<table class="table table-bordered">
  ...
</table>

Or one with striped rows: 


<table class="table table-striped">
  ...
</table>

Step 2

In addition to styling the whole table, we can also create hover states for each row: 


<table class="table table-hover">
  ...
</table>

Contextual colors can also be applied to each tr.  This also works with th and td: 


<table class="table">
  ...
    <th class="warning">
  ...
  <tr class="info">
  ...
    <td class="danger">
  ...
</table>

Step 3

Using the input box below, practice styling the example table:
(Note: you can style the text and background as well) 


<table class='{{tablePractice}}'>
  ...
</table>
