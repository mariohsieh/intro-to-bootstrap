Tables

Goals

  • Add different classes to style tables

  • Style rows with hover states and contextual colors

Overview

This lesson we'll go through the different table styles that Bootstrap offers.  In addition, we'll see how to style rows with contextual colors.

Steps

Step 1

Tables can look much better with only a couple added Bootstrap classes.  A basic table with no added CSS looks like this:

By adding one class table:

We can create a bordered table:

<table class="table table-striped">

</table>
